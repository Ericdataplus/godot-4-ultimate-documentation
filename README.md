## ⚡ Quick Start

New to Godot? Read these in order:
1. Start with [GDScript Mastery](./01-gdscript-mastery.md) to learn the language
2. Read [Scene Architecture](./02-scene-architecture.md) to understand Godot's philosophy
3. Read [**Modular Architecture**](./19-modular-architecture.md) to structure your project for real-world scale
4. Pick a topic relevant to your project (Physics, UI, 3D, etc.)
5. Check [Common Pitfalls](./16-common-pitfalls.md) when you hit a wall
6. Browse [Tips & Tricks](./17-tips-and-tricks.md) to level up
